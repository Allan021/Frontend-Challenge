# Frontend-Challenge

Aplicacion para muestra de itens por busqueda

## Iniciar la aplicacion con yarn (recomendable)
Instalar las dependencias
```bash
yarn
```
Ejecutar el entorno local
```bash
yarn dev
```
## Iniciar la aplicacion con npm
Instalar las dependencias
```bash
npm install
```
Ejecutar el entorno local
```bash
npm run dev
```
